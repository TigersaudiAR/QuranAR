# Quran Page Images

This directory should contain Quran page images named from `001.jpg` to `604.jpg`.

## Where to Get Images

You can obtain high-quality Quran page images from:

1. **Mushaf Madina** - Available from King Fahd Complex for the Printing of the Holy Quran
2. **Quran.com** - Offers downloadable page images
3. **Tanzil Project** - Provides various Mushaf images
4. **Hafs Uthmani Script** - Available from multiple Islamic resources

## File Naming Convention

Files must be named with zero-padded three-digit numbers:
- Page 1: `001.jpg`
- Page 2: `002.jpg`
- ...
- Page 604: `604.jpg`

## Image Requirements

- Format: JPEG (.jpg)
- Recommended resolution: 1200x1800 pixels or higher
- High quality scans for better readability
- Clear and crisp text

## Fallback

If images are not available, the Mushaf viewer will display a placeholder image.
You can create a `placeholder-quran-page.jpg` in the `/frontend/public/` directory as a fallback.
